Langage Mot clé Utilité
HTML html C’est l’élément racine du document. Tout le contenu de la page doit être à l’intérieur de cette balise.
head Contient tout le contenu visible : texte, images, boutons, tableaux, etc.
a Permet de créer un lien hypertexte. On utilise l’attribut href pour indiquer l’adresse de destination.
div Conteneur générique servant à regrouper des éléments pour organiser la page ou appliquer du style.
body Contient tout le contenu visible : texte, images, boutons, tableaux, etc.
CSS color Définit la couleur du texte.
background-color Définit la couleur d’arrière-plan d’un élément.
display Détermine la manière dont un élément est affiché (block, inline, flex, none, etc.).
margin Définit l’espace extérieur autour d’un élément.
padding Définit l’espace intérieur entre le contenu et la bordure.
Javascript let Déclare une variable dont la valeur peut être modifiée.
const Déclare une variable constante (sa valeur ne peut pas être changée).
function Permet de définir une fonction, c’est-à-dire un bloc de code réutilisable.
if Permet de créer une condition (exécuter du code seulement si une condition est vraie).
document Objet qui représente la page web et permet de modifier son contenu.
